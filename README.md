# Pizza-Sales-Data-Analysis-Dashboard
In real-world businesses, understanding sales performance is critical for decision-making. This project simulates a food retail analytics scenario where pizza sales data is analyzed to uncover trends, customer preferences, and revenue drivers.

The project covers the full data analytics pipeline:

Raw Data → SQL Analysis → KPI Extraction → Excel Dashboard → Business Insights
